
💼 Freelance Portfolio Website

A personal freelancing website to showcase services, portfolio projects, and accept client payments.

📌 Overview
A fully responsive single-page freelancing portfolio website built with pure HTML, CSS, and JavaScript. Designed to help freelancers reach clients, present their work, and receive payments — all in one place.

🚀 Services Showcased

Web Development
UI/UX Design
Mobile App Development
API Integration
SEO & Performance Optimization
Monthly Maintenance


💳 Payment Integration

Paytm deep-link integration
UPI ID copy to clipboard
Bank transfer via WhatsApp
Discuss-first option before payment


✨ Features

Single file — entire site in one index.html
Fully responsive (mobile + desktop)
Dark theme with smooth animations
Scroll-triggered fade-in effects
Auto-hiding navbar on scroll down
Payment modal with multiple options
Contact form via mailto
WhatsApp quick-link integration
Toast notifications for user feedback


🛠️ Built With

HTML5, CSS3, Vanilla JavaScript
Google Fonts — Syne + DM Sans
No frameworks, no dependencies, no build step
